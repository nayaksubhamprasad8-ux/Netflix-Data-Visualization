🎬 Netflix Data Visualization Project








📌 Overview

This project focuses on analyzing and visualizing Netflix dataset to understand content trends, distribution, and patterns. The dataset is cleaned and processed using Python, and multiple visualizations are created to extract meaningful insights.

🚀 Features

Data cleaning and preprocessing
Movies vs TV Shows comparison
Year-wise content trend analysis
Movie duration distribution
Content rating analysis
Top 10 countries by content

🛠️ Tech Stack

Python
Pandas
Matplotlib

📊 Visualizations

1. Movies vs TV Shows

2. Movies & TV Shows Released Each Year

3. Movie Duration Distribution

4. Rating Distribution

5. Release Year Trend

6. Top 10 Countries

📈 Key Insights

Movies are more in number compared to TV Shows
Content increased rapidly after 2015
Most movies have duration around 80–120 minutes
Majority of content is rated TV-MA and TV-14
United States contributes the highest content
